# STEP It Out — UI Design

The interface should feel like a professional sequencer similar to Ableton Live or FL Studio.

## Layout

TOP BAR
Playback and timing controls

MAIN GRID
Layer controls and sequencing

PREVIEW MONITOR
Lighting preview

## Top Bar

Loop Length  
FPS  
Play / Stop  
Current Frame  
Current Time

The timeline must be a **ruler**, not a slider.

## Layer Row

Each row contains:

thumbnail  
layer name  
division control  
attack  
decay  
curve  
step grid

## Thumbnails

Must preserve aspect ratio.

thumbnail height = 48px

No stretching.
